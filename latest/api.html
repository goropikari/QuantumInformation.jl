QuantumInformation.ghz — Function.
   ghz(n)

n qubits GHZ(Greenberger–Horne–Zeilinger) state

\[| \mathrm{GHZ} \rangle = \frac{| 0 \rangle^{\otimes n} + | 1 \rangle^{\otimes n}}{\sqrt{2}}\]

Example

julia> n = 3;

julia> ghz(n)
Ket(dim=8)
  basis: [Spin(1/2) ⊗ Spin(1/2) ⊗ Spin(1/2)]

      0.0+0.0im
      0.0+0.0im
      0.0+0.0im
      0.0+0.0im
      0.0+0.0im
      0.0+0.0im
 0.707107+0.0im

QuantumInformation.phaseshift — Function.
   phaseshift(λ)

Phase shift gate. λ must be real number. This is equivalent to singleunitary(0, 0, λ).

\[U(\lambda) = \begin{bmatrix} 1 & 0 \\ 0 & e^{i \lambda} \end{bmatrix}\]
source
QuantumInformation.singleunitary — Function.
   singleunitary(θ, ϕ, λ)

Single qubit unitary gate.

\[U(\theta, \phi, \lambda) = \begin{bmatrix} \cos(\theta/2) & -e^{i \lambda}\sin(\theta/2) \\ e^{i \phi}\sin(\theta/2) & e^{i (\lambda + \phi)} \cos(\theta/2) \end{bmatrix}\]

QuantumInformation.controlsgate — Function.
   controlsgate(N, controls, target, op)

Multiple controlled gate.

Arguments

  • N::Int: the number of qubits.

  • controls::Vector{Int}: list of controlled qubit positions.

  • target::Int: position of target qubit.

  • op::Operator: Operator that acts on target qubit.

Example

julia> nqubits, controls, target, op = 3, [1, 2], 3, sigmax();

julia> controlsgate(nqubits, controls, target, op) # equivalent to toffoli gate
SparseOperator(dim=8x8)
  basis: [Spin(1/2) ⊗ Spin(1/2) ⊗ Spin(1/2)]

  [5, 5]  =  1.0+0.0im
  [6, 6]  =  1.0+0.0im
  [8, 7]  =  1.0+0.0im

QuantumInformation.tex — Function.
tex(x, statename="ψ") # for Ket and Bra
tex(x, statename="Operator") # for Operator

LaTeX (ASCII art) rendering for qubits system's Ket, Bra, and Operator in IJulia (REPL). You can change LHS state name by second argument.

Example (REPL)

julia> id = identityoperator(SpinBasis(1//2));

julia> ψ = cnot() * (hadamard() ⊗ id) * qubit("00")

julia> tex(ψ)
|ψ> = 0.707|00> + 0.707|11>

julia> dm(ψ)
DenseOperator(dim=4x4)
  basis: [Spin(1/2) ⊗ Spin(1/2)]
 0.5+0.0im  0.0+0.0im  0.0+0.0im  0.5+0.0im
 0.0+0.0im  0.0+0.0im  0.0+0.0im  0.0+0.0im
 0.0+0.0im  0.0+0.0im  0.0+0.0im  0.0+0.0im
 0.5+0.0im  0.0+0.0im  0.0+0.0im  0.5+0.0im

julia> dm(ψ) |> tex
Operator = 0.5 |00><00| +0.5 |00><11| +0.5 |11><00| +0.5 |11><11|
